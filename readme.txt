微信小程序与云开发入门项目：最新电影推荐小程序
________________________________________________________
--------------------------------------------------------

主要功能：

- 通过在云函数中调用豆瓣电影API接口获取电影信息列表并展示；
- 对电影进行评价（打分+上传图片）；
- 评价图片上传到云存储，评价内容保存到云数据库；
- 获取当前小程序用户信息。

---------------------------------------------------------

核心知识点：

- 使用Promise处理JS异步问题；
- 对云数据库进行增删改查的操作及权限管理；
- 创建云函数及通过云函数实现获取用户openid、操作云数据库；
- 实现文件在云存储中的上传、下载、展示。
